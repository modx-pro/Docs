# Remains page

Besides editing remains in the product form, you can manage stock on the component page: **Applications → miniShop2 → Remains** in the manager top menu.

![Remains page](https://file.modx.pro/files/6/7/8/6782f3a9e0bb917e6bb5ac150027fd12.png)

This table lists all remains records, including obsolete attribute combinations. You can delete such rows if they are no longer needed.

To edit remains, double-click the value you want to change; an input field will appear. Enter the new quantity and press Enter to save it to the database.
