# msOptionsPrice2

Product options

## Standard fields

| Field        | Name             | Possible values |
| -------------| -------------------- |------------------- |
| name         | Modification name |                    |
| rid          | id resource           |                    |
| type         | Modification type      | `1 \|\| 2 \|\| 3`  |
| price        | Price                 |                    |
| old_price    | Old price          |                    |
| article      | SKU              |                    |
| weight       | Weight                  |                    |
| count        | Quantity           |                    |
| image        | Image             |                    |
| active       | Active modification | `0 \|\| 1`         |
| modification | Modification options    |                    |
