# Products (msProduct)

## Export fields

All resource fields plus custom fields

| Field           | Name                                      |
| --------------- | --------------------------------------------- |
| price           | Price                                          |
| old_price       | Old price                                   |
| article         | SKU                                       |
| weight          | Weight                                           |
| color           | Colors                                 |
| size            | Sizes                               |
| vendor          | Vendor                                 |
| made_in         | Country                                        |
| tags            | Tags                                  |
| new             | New                                         |
| favorite        | Special                                        |
| popular         | Popular                                    |
| categories      | Product categories (id)                         |
| categories_name | Product categories (name)                       |
| images          | Image list. ms2Gallery supported |
