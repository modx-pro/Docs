# Snippet GoogleSheetsFront

Snippet runs export/import from the front-end

## Parameters snippet

| Field | Description        | Values                                           |
| ----- | ------------------ | ------------------------------------------------ |
| id    | Export/import ID   |                                                  |
| mode  | Mode               | export \|\| import \|\| migxExport \|\| migxImport |

## Example

```modx
[[!GoogleSheetsFront?
  &mode=`export`
  &id=`5`
]]
```
