# List field settings

You can show only the fields you select via the button.

![List field settings](https://file.modx.pro/files/f/2/d/f2d5a31fda99b0c046976be851cddc3b.png)

A modal opens where you can search for the field.

To add a field, drag it to the right column.

You can also set the column width for a field.

![List field settings](https://file.modx.pro/files/0/c/7/0c76082ca748a3d5c99ef228a5ac5d2e.png)

## Template field selection

You can add TV parameters that have template access.

![Template field selection](https://file.modx.pro/files/3/e/b/3eb2f646a64e2987e26773a747faa44a.png)

### Category field selection

You can add options bound to a category.

![Category field selection](https://file.modx.pro/files/0/c/3/0c39b4dac32229d8f937cdec0e7da180.png)

### Field position

Use the arrows to change field order. Select a field and drag it up.

![Field position](https://file.modx.pro/files/e/c/3/ec3519e7799d005daee333c289d2ff96.png)

### Setting width

Double-click the numbers to set column width.

![Setting width](https://file.modx.pro/files/b/4/d/b4d7df88e3ac95ad0d17178ca8b83896.png)
