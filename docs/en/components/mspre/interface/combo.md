# COMBO menu

This menu contains bulk actions for setting new values for certain data types.

![This menu contains](https://file.modx.pro/files/d/6/c/d6cb85402b9ad77a5d7b31c327fa810d.png)

## Change template

Select resources on the page and choose **Change template**

![Change template](https://file.modx.pro/files/1/2/3/123178a08a2140077b306a1bf6fbb6b3.png)

## Change file source

Select resources on the page and choose **Change file source**

::: warning
File source change is for products only
:::

![Change file source](https://file.modx.pro/files/a/1/e/a1e73bac3131189e051e979a2ba23bb0.png)

## Change vendor

Select resources on the page and choose **Change vendor**

::: warning
Vendor change is for products only
:::

![Change vendor](https://file.modx.pro/files/9/f/8/9f8145aa051a55f84ecd397f9c62388f.png)

## Change parent

Move resources from one parent (category) to another.

Select resources on the page and choose **Change parent**

![Change parent](https://file.modx.pro/files/c/6/6/c6635fe5ccce5bb7695a76899c340027.png)

## Change content type

Change content type, e.g. from html to pdf.

Select resources on the page and choose **Change content type**

![](https://file.modx.pro/files/1/7/5/1750c6b6b89327b3454ba25d1ae96650.png)

## Assign resource group

Bulk assign a resource group to selected resources.

![Assign resource group](https://file.modx.pro/files/0/2/0/020b416221dbc8f536386e871bb8bf52.png)

Select resources on the page and choose **Assign resource group**

![Select resources on the page](https://file.modx.pro/files/4/7/5/47531c5ae7dacfdcf8eed1b569443883.png)

## Change dates

Change dates for resources.

Fields available:

- Creation date **createdon**
- Edit date **editedon**
- Published **publishedon**
- Publish date **pub_date**
- Unpublish date **unpub_date**

Select resources on the page and choose **Change dates**

![Select resources and choose](https://file.modx.pro/files/5/3/0/530123930da430e921cbaa30a23fb6dd.png)

## Change user

Change user for resources.

Fields available:

- Created by **createdby**
- Edited by **editedby**
- Unpublished by **publishedby**

Select resources on the page and choose **Change user**

![Select resources and choose](https://file.modx.pro/files/f/3/a/f3a0d20865dea3a8c666a152ce85a90a.png)

## Set text values

Set text values for fields:

- Title **pagetitle**
- Long title **longtitle**
- Menu item **menutitle**
- Link attributes **link_attributes**
- Description **description**
- Intro **introtext**

Select resources on the page and choose **Set text values**

Choose the field to set.

Enter the text in the text field.

Check "Leave empty" to clear the field.

![If you want to clear the field](https://file.modx.pro/files/5/8/5/5855b628c7c13beefe7541201eac176e.png)
