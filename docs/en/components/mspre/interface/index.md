## Bulk actions console

A special console runs during each bulk action to show processing progress.

This lets the component process as many resources as you select on the page.

![component can process](https://file.modx.pro/files/2/c/7/2c72e09cb959a0c090a8b27752517bdf.png)

While processing, the numbers next to the ID change; that is the ID of the resource being processed.
