# List editing

Most fields have built-in handlers that open field editors so you don't need to run bulk actions.

Click in the field area for the resource

![Click in the field area for the resource](https://file.modx.pro/files/4/0/6/406d8e3d02910f5f1d4894a78d5d3a9d.png)

## Category binding and template access

Some options and TV parameters may not be bound to a category or may not have template access.
A special mechanism checks option or TV availability before opening the edit modal.

## Editing TV parameters

### Images

To edit a TV with input type **Image**, double-click the field area for the resource.

A modal will open where you select the image

![Images](https://file.modx.pro/files/c/7/f/c7ffeb8db9a83f8e812cd23c1a4b96d9.png)

File manager opens

![Images](https://file.modx.pro/files/8/7/a/87a523add8d4914ec3697857a020026b.png)

Select the image

![Images](https://file.modx.pro/files/a/c/9/ac9b34fdce5d22108372e03aaed65d52.png)

And click save

![Images](https://file.modx.pro/files/0/5/2/052be935f304bf8a2858784bc26ef006.png)

## Input types

### Standard and custom fields

Handlers for standard fields were set manually; check the demo for edit capability.

::: danger
Not all fields can be edited. If a field doesn't edit, check the demo. If it doesn't edit there either, it's not editable.
:::

[**Demo**](http://demo.mspre.bustep.ru/manager/?a=resource&namespace=mspre)

- Login: `manager`
- Password: `manager`

### TV parameters

Modal opens with input type according to **Input type**

![TV parameters](https://file.modx.pro/files/7/3/c/73cbd6ee3bb6e444489cba32f352b1ff.png)

::: danger
Editing is supported only for standard TV parameters shipped with MODX. If a field doesn't edit, ensure it's not custom and not installed by another component.
:::
