# What the component does

1. Convenient and fast management of large numbers of resources without opening the resource card
2. Managing discounts and prices for products
3. Finding resources in the database using filters
4. Saving content managers hundreds of hours on resource editing
5. Quick view of resource field contents via column management to find and fix errors in descriptions and parameters

## Main capabilities

- Bulk set/replace/delete values for **TV parameters** (Additional fields)
- Bulk set/replace/delete values for **miniShop2 options**
- Bulk change template, parent, resource type, resource group assignment, date, users
- Bulk set values for text fields: **pagetitle, longtitle, menutitle, link_attributes, description, introtext**
- Bulk set flags: **published, deleted, hide in menu, searchable, show in tree, cacheable, freeze URL, use html editor, hide children in tree**
- Bulk set flags for products: **new, popular, special**
- Bulk set additional categories for products
- Bulk **image preview** generation for products
- Bulk actions on selected resources: **clear cache, generate URL, delete resources (requires delete flag before deletion)**
- Set values by clicking the field in the column for a resource; edit a single resource without opening the product card. Almost any fields, **TV parameters**, **miniShop2 options** can be changed. Same input types are used as when editing via the card
- Export resources to xls and CSV (option to include TV parameters and miniShop2 options). Field configuration for export is done right in the component
- **Price** management with percent increase/decrease or new value. Ability to copy one price column to another
- Transaction logging for price field changes with ability to undo
- Resource filtering via parent tree and resource filter panel:
  - **Statuses** — Enabled/Disabled, Deleted/Not deleted, New/Not new, Popular/Not popular, Special/Not special, With image/Without image, URL duplicates, Show hidden in tree
  - **Templates** — sort by templates
  - **Vendor** — sort by vendor
  - **Filter field** — specify field for filtering and operator
  - **Show nested resources** — show all children of the resource
  - **Show products from additional categories** — show all products from miniShop2 additional categories
  - **Class** — sort by resource class_key
- Column management for standard resource fields and additional fields like **TV parameters** and **miniShop2 options** with value rendering for display
