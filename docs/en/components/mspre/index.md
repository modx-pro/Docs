# msPre

The component is for bulk management of resources/products in MODX Revolution, supporting versions from 2.3 through MODX 3.
You can manage thousands of resources at once through the component.

Successfully tested on a database with **120,000 products**.
Also tested on MODX 3 which is in development; no issues observed.

[Demo version](http://demo.mspre.bustep.ru/manager/?a=resource&namespace=mspre)
