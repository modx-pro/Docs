# CitySelect

Component for city selection. City suggestions while typing use kladr-api. User geolocation uses SxGeo.

## Module settings

To edit script settings go to **System settings** and filter by "cityselect".

![Module settings - 1](https://file.modx.pro/files/a/d/9/ad9bb66a4c426a1ff3e686273d447abd.png)

- kladr-api key — optional
- SxGeo database name
- kladr-api token — optional

## Snippet call

On install an example chunk is available; it is used by default when you call the snippet.
