# Application conditions

For any single coupon not linked to a promotion, you can specify when the promo code discount applies to the order.

[![](https://modx.pro/assets/images/tickets/8405/24346f91041dddc1f6da9f560611f334dddf7b82.png)](https://modx.pro/assets/images/tickets/8405/24346f91041dddc1f6da9f560611f334dddf7b82.png)

## Condition list

- Cart total (from)
- Cart total (to)
- Product count in cart (from)
- Product count in cart (to)

Each condition can be used alone or combined.
