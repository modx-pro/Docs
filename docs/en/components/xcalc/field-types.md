# Field types

The component has several field types, each with its own behavior.

## Radio

Single choice. Often the most used type in calculators. msProducts type is partly built on it.

## Checkbox

Multiple choice or no value. Can be used for extra options.

## msProducts

Outputs a product list with one selectable for use in price calculation.

## Number

Numeric field. For length/width/height/quantity etc.

## Text

Text field. For comments or similar input.
