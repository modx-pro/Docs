# Manager

The Manager is on the resource page, available from the menu and with the [[Alt]]+[[g]] shortcut.
It has the following tabs:

## Main

Tab that shows the loaded [data][2].
Actions: **load**/**create**/**edit** data.

![Main](https://file.modx.pro/files/1/e/9/1e9d05d759ccf7a3abfc7b28c4746b95.png)

## Snippets

Tab that lists [snippets][3].
Actions: **create**/**edit**/**delete** snippet.
