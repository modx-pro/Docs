# Settings

ResourceGrabber settings are in **System settings** > **ResourceGrabber**

## Main

Main settings:

- `{working_templates}` — active templates. Comma-separated list of template IDs for which to enable the feature.
- `{fields_on_create}` — list of fields available when creating a resource. Default is empty — all fields are available.
- `{fields_on_update}` — list of fields available when updating a resource. Default is "currency,price,old_price".
