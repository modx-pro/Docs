# Data

## Load / Create / Edit data

Data from the specified URL will be loaded and processed by the snippet.

![Load / Create / Edit data](https://file.modx.pro/files/9/6/1/961f24958e221fbfb5109f0bc56a2a61.png)

The selected `type` adjusts the loaded data according to the settings:

- `fields_on_create` — fields available when creating.
- `fields_on_update` — fields available when updating.
