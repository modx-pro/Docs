# textAdvs

**textAdvs** is a component for embedding ad blocks into resource `content` when the page is displayed to the user.

## Features

- Place ads before/after a given tag (by position),
- Run only on a specific template,
- Show ad blocks one after another (as the page is updated),
- Use custom tags.
