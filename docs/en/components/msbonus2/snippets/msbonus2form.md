# msBonus2Form

Outputs the bonus form. Call the snippet on the miniShop2 cart page.

The snippet runs only for an authenticated user in the current context.

## Parameters

| Name   | Default             | Description          |
|--------|----------------------|----------------------|
| **tpl**| `tpl.msBonus2.form` | Chunk for the form.  |

## Example

```fenom
{'!msBonus2Form' | snippet}
```
