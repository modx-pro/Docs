# UserAuthHash events

Available events:

* `uahOnBeforeGetAuthHash`
* `uahOnGetAuthHash` - auth hash retrieval
  *`object` - hash object
  *`user` - user object
* `uahOnBeforeProcessAuthHash`
* `uahOnProcessAuthHash` - hash processing
  *`object` - hash object
  *`user` - user object
* `uahOnBeforeRemoveAuthHash`
* `uahOnRemoveAuthHash` - hash removal
  *`user` - user object
