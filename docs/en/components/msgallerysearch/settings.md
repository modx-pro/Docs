# System settings

Extra component settings for convenience:

| Parameter                               | Default            | Description                        |
| --------------------------------------- | ------------------ | ---------------------------------- |
| **msgallerysearch_available_buttons**  | download,google,gallery | Buttons on the panel          |
| **msgallerysearch_copy_meta**          | Yes                 | Copy image metadata                |
| **msgallerysearch_custom_search**      | Empty               | Custom search URL                  |
| **msgallerysearch_disable_auto_query** | Yes                 | Disable auto-loading of results    |
| **msgallerysearch_disable_minishop2**  | No                  | Disable for miniShop2              |
| **msgallerysearch_disable_ms2gallery** | No                  | Disable for ms2Gallery             |

Button names for registration:

- download — load from link
- google — find in Google
- gallery — find in images
