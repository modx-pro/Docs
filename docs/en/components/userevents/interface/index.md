# Interface

The main Manager is in a separate site section, available from the menu.
It has the following tabs:

## Events

Tab that lists [events][4]. Sortable and searchable.
Actions: **create**/**edit**/**delete** event.

![Events](https://file.modx.pro/files/2/9/e/29ef5c704b769c501e91c6f9021abd00.jpg)

## Statuses

Tab that lists [statuses][8].
Actions: **create**/**edit**/**delete** status.

![Statuses](https://file.modx.pro/files/2/6/2/262a170102ccddd6bcefad4bc45fc195.jpg)
