# ms3RecentlyViewed snippets

The component provides three snippets: viewed block, similar products block, and frontend lexicon.

- [ms3recentlyviewed](ms3recentlyviewed) — output list of products by given IDs (or from DB when fromDB=true)
- [ms3recentlyviewedSimilar](ms3recentlyviewedSimilar) — products from the same categories as viewed (“similar to viewed”)
- [ms3rvLexiconScript](ms3rvLexiconScript) — lexicon and config for JS (language, maxItems); include **before** viewed.js
