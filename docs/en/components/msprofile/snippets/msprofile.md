# msProfile

Snippet for outputting user profile data on the frontend.

| Name   | Default | Description                                                                                                           |
|--------|---------|-----------------------------------------------------------------------------------------------------------------------|
| **&id**| `0`     | User ID for profile output; if not specified, the current user is used.                                               |
| **&tpl**|        | Chunk for user profile output. You can use @INLINE chunks without filters and snippets.                               |

The snippet has no default chunk, so when called without one you will see all available placeholders and their values.
