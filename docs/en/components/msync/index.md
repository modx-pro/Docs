# mSync

The component imports products and offers from **1C, Class365 and MoiSklad** via **CommerceML**
into a **miniShop2** store.

Exchange protocol description: [here](http://v8.1c.ru/edi/edi_stnd/131/#1).

[Class365 integration guide](http://class365.ru/help/modx) (for Class365 support contact Class365).
