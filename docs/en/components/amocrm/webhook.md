# Webhook setup

To receive updates from amoCRM, in the **Webhook** section enter:
`<http://site.ru>/assets/components/amocrm/webhook.php`
Replace `<http://site.ru>` with your site URL.

![Webhook setup - 1](https://file.modx.pro/files/f/8/c/f8cc515579f8ba122c7ae846d8698b81.png)

Then in the dropdown on the right enable:

- Update deal
- Deal status change
- Add contact
- Update contact
