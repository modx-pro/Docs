# msPointsIssue snippets

All msPointsIssue snippets work via pdoTools and expect [Fenom][010103] in chunks.

This allows:

- reducing the total number of chunks
- improving workflow
- faster execution
- more complex chunks with advanced condition checks via Fenom functions
