# msPointsIssue settings

msPointsIssue settings are in **System settings** > **msPointsIssue**

## General

Main settings:

- `{front_css}` — CSS file to load on the frontend.
- `{front_js}` — JavaScript file to load on the frontend.
- `{map_service}` — Map service. Options: "google", "yandex".
- `{google_maps_script}` — Path to Google Maps script.
- `{yandex_maps_script}` — Path to Yandex Maps script.

## Geonames

Settings for [geonames](http://www.geonames.org):

- `{geonames_api_url}` — URL for GeoNames API requests.
- `{geonames_api_username}` — Username for GeoNames API.
- `{geonames_country}` — Country for GeoNames API requests.
- `{geonames_continent_code}` — Continent code for GeoNames API requests.
- `{geonames_api_limit}` — Limit for GeoNames API requests.
