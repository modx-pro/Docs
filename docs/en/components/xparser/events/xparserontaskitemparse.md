# xParserOnTaskItemParse

Fires after processing the field list. Before fetching data from a subtask, if one exists.

## Parameters

- `xParser $xp` — main component class instance,
- `xpParser $parser` — parser class instance,
- `array $task` — task array,
- `string $item` — raw article code string,
- `array $row` — article fields array.
