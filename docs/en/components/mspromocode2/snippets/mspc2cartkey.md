# mspc2CartKey

Due to how the cart works, helper snippet `mspc2CartKey` was added. It is used only as a modifier.

::: warning
Not used as a standalone snippet.
:::

## Parameters

| Name    | Default | Description                                                                         |
|---------|---------|-------------------------------------------------------------------------------------|
| **input**|         | Product data array. Must contain `id` and `options`.                                 |

See [Quick start][02] for usage.
