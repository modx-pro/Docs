# Snippets

## mscdekConnector

### Purpose

Handles component requests on the server for status, pickup point list, and hints.

### Parameters

The snippet reads parameters from presets. The presets file path can be set in system setting `ms_cdek2_presets_path`.

### Output

Returns JSON depending on the preset passed.

### Usage

The snippet is used internally by the component and is not intended for direct use.
