# handyShop

![handyShop](https://file.modx.pro/files/c/e/a/cea37261ae548771e9f8b1ef3c2b4f4d.png)

> Updated 08.02.2023, current package version v1.0.0 beta1

HandyShop is a ready-made theme for an online store or catalog.
It lets you build a catalog or online store based on the miniShop2 package without diving deep into MODX Revolution programming.
