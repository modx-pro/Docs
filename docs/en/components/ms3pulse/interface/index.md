# ms3Pulse interface

Short overview of interface sections. Details are on the linked pages.

## Entry points

| Section | Access | Description |
|--------|--------|----------|
| **Dashboard** | Components → ms3Pulse | Main page: metrics, filters, tabs with charts and tables. |
| **Builder** | ms3Pulse → Builder | Add custom charts to Dynamics and Products tabs. |
| **Settings** | ms3Pulse → Settings | System settings (namespace ms3pulse). |

## Views

The controller supports three views via `?view=`:

- **dashboard** (default) — main dashboard.
- **builder** — chart builder page.
- **topflops** — Tops / Flops page (also available as a dashboard tab).

## Documentation by section

- [Dashboard](dashboard) — header, filters panel, tabs, empty states.
- [Chart builder](builder) — add chart form, preview, localStorage.
- [Tops and flops](topflops) — table types, add, sort, pagination, export.
- [Settings](settings) — shortcut to ms3pulse system settings.
