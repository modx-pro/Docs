# Clients

## Create client

![Create client](https://file.modx.pro/files/5/4/6/546c1aa61ab8a310c0beda6e6d162de6.png)

To create a client select a site user or enter an email. If no user with that email exists, one will be created automatically.
In the `client_groups` setting you can list user groups, comma-separated.

## Edit client

![Edit client](https://file.modx.pro/files/8/7/1/8719151594fef6d34f07b6dbf1dc8c5a.png)

Client status can be edited. The [subscriptions][7] tab is available for editing.
