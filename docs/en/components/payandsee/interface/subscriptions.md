# Subscriptions

## Create subscription

![Create subscription](https://file.modx.pro/files/b/a/2/ba294b5e5f5a03048ec1986788f50dc5.png)

To create a subscription select the content and site user. Set the subscription start and end date.

## Edit subscription

![Edit subscription](https://file.modx.pro/files/4/8/a/48a40c0b7b1e1b48f2431905f39cf49e.png)

Content, client and subscription status can be edited. You can adjust the subscription period.
