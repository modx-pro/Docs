# Interface

The main Manager is in a separate site section, available from the menu.
It has the following tabs:

## Content

Tab that lists [content][4]. Sortable by content status and search.
Actions: **create**/**edit**/**delete** content.

![Content](https://file.modx.pro/files/d/1/5/d15024f579213627921a4162060baee4.png)

The set and order of fields on this tab is controlled by the system setting **grid_content_fields**.

## Rates

Tab that lists content [rates][5]. Sortable by content, rate validity and search.
Actions: **create**/**edit**/**delete** rate.

![Rates](https://file.modx.pro/files/d/b/9/db955e0e779bae418f508be6a32ee2fa.png)

The set and order of fields on this tab is controlled by the system setting **grid_rate_fields**.

## Clients

Tab that lists [clients][6]. Sortable by status and search.
Actions: **create**/**edit**/**delete** client.

![Clients](https://file.modx.pro/files/4/2/9/42970ae76d6fcdf0e780247cef891416.png)

The set and order of fields on this tab is controlled by the system setting **grid_client_fields**.

## Subscriptions

Tab that lists [subscriptions][7]. Sortable by client, content, status and search.
Actions: **create**/**edit**/**delete** subscription.

![Subscriptions](https://file.modx.pro/files/0/4/4/044adff1dc3931af547c3058fd49cbc4.png)

The set and order of fields on this tab is controlled by the system setting **grid_subscription_fields**.

## Statuses

Tab that lists [statuses][8]. Sortable by class.
Actions: **create**/**edit**/**delete** status.

![Statuses](https://file.modx.pro/files/c/c/5/cc5f4d4a829b4b3989945a1c0fcdfb37.png)

The set and order of fields on this tab is controlled by the system setting **grid_status_fields**.

## Notifications

Tab that lists [notifications][9]. Sortable by class and search.
Actions: **create**/**edit**/**delete** notification.

![Notifications](https://file.modx.pro/files/9/5/6/956b77954e2bd2b53dd24a9e4f491fe5.png)

The set and order of fields on this tab is controlled by the system setting **grid_alert_fields**.
