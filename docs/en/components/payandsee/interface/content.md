# Content

## Create content

![Create content](https://file.modx.pro/files/3/a/3/3a32615ed16bb62669d402d487279204.png)

To create content you must specify the resource it will be linked to. Content rates and subscriptions become available after saving.

## Edit content

![Edit content](https://file.modx.pro/files/2/c/3/2c3b3e6bef004be17241ca0bc590ff65.png)

All content fields can be edited; the [rates][5] and [subscriptions][7] tabs are also available.
