# Скриншоты для документации ms3Pulse

В эту папку нужно поместить скриншоты с указанными ниже именами. В документации используются пути вида `/components/ms3pulse/screenshots/имя-файла.png`.

## Список файлов (заглушки для вставки)

| Файл | Описание (что должно быть на скриншоте) |
|------|----------------------------------------|
| `menu-ms3pulse.png` | Пункт меню «Компоненты → ms3Pulse» |
| `dashboard-overview.png` | Главный экран дашборда — вкладка «Обзор» |
| `filters-panel.png` | Панель фильтров с периодом и статусами (раскрытая) |
| `tab-dynamics.png` | Вкладка «Динамика» с графиками |
| `builder-add-chart.png` | Конструктор графиков — форма «Добавить график» |
| `builder-page.png` | Страница конструктора с превью типов графиков |
| `builder-form.png` | Форма «Добавить график» с выбранным типом и источником |
| `dashboard-header.png` | Шапка дашборда с бейджем и кнопкой тура |
| `dashboard-filters.png` | Раскрытая панель фильтров |
| `tab-overview-metrics.png` | Вкладка «Обзор» — карточки метрик |
| `topflops-tables.png` | Вкладка «Топы / Флопы» с таблицами |
| `topflops-add-table.png` | Форма «Добавить таблицу» |
| `settings-namespace.png` | Системные настройки — пространство ms3pulse |
| `metrics-cards.png` | Карточки метрик на вкладке «Обзор» |
| `chart-types-examples.png` | Примеры нескольких типов графиков |
| `export-csv-button.png` | Кнопка экспорта CSV у таблицы |
| `export-chart-menu.png` | Меню карточки графика с пунктом экспорта PNG |
| `scheduler-task.png` | Задача Scheduler с процессором SendScheduledReport |
| `permissions-policy.png` | Права доступа — политика с правом ms3pulse_view |

После добавления изображений удалите этот README или оставьте для справки.
