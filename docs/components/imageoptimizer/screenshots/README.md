# Скриншоты админки ImageOptimizer

Положите PNG в эту папку. В документации пути вида `/components/imageoptimizer/screenshots/имя-файла.png`.

После добавления изображений раскомментируйте строки `![...](...)` в [manager.md](../manager.md) и [quick-start.md](../quick-start.md).

| Файл | Что снять |
| --- | --- |
| `menu-imageoptimizer.png` | Пункт меню **Компоненты → ImageOptimizer** |
| `overview.png` | Вкладка **Обзор**: статистика очереди, прогресс, кнопки **Обработать очередь** / **Сбросить зависшие** |
| `queue.png` | Вкладка **Очередь**: таблица, фильтр по статусу, поиск по пути |
| `queue-rebuild.png` | Диалог **Пересобрать очередь**: media source, path, dry-run |
| `settings.png` | Вкладка **Настройки**: форма с группами (форматы, фронтенд, очередь) |
| `server.png` | Вкладка **Server**: PHP, лимиты, карточки энкодеров WebP/AVIF, cron-команда |
| `compatibility.png` | Вкладка **Compatibility**: VueTools, Thumb3x, MS3 |
