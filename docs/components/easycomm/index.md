# easyComm

Компонент easyComm предназначен для создания на сайтах таких разделов и блоков, как:

- отзывы;
- комментарии;
- вопросы пользователей.

Компонент можно использовать на любых страницах сайта, например в карточках товара или на странице О компании.

Отличительной особенностью easyComm является встроеная поддержка пользовательских оценок, которые позволяют при создании отзывов оценить что-либо по указанной шкале (напр. от 1 до 5), а в дальнейшем вывести средний рейтинг.

Компонент работает как в MODX Revolution версии > 2.6, так и в MODX 3. В магазине дополнений находятся 2 версии установочного пакета, нужная будет скачана автоматически в зависимости от вашей версии MODX.
