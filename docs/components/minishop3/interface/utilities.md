# Утилиты

Раздел утилит предоставляет инструменты для массовых операций и настройки компонента MiniShop3.

## Доступ

**Меню:** Приложения → MiniShop3 → Утилиты

## Вкладки

| Вкладка | Описание |
|---------|----------|
| [Галерея](utilities/gallery) | Массовая перегенерация миниатюр изображений |
| [Импорт](utilities/import) | Импорт товаров из CSV-файлов |
| [Поля товара](utilities/product-fields) | Настройка отображения полей в карточке товара |
| [Дополнительные поля](utilities/extra-fields) | Создание новых полей для моделей |
| [Колонки гридов](utilities/grid-columns) | Настройка колонок в таблицах |
| [Поля модели](utilities/model-fields) | Управление полями моделей из БД |

## Обзор возможностей

### Галерея

Инструмент для массовой перегенерации миниатюр изображений товаров. Полезен после:
- Изменения настроек превью в системных настройках
- Миграции сайта
- Массового обновления изображений

### Импорт

Пошаговый мастер импорта товаров из CSV-файлов с возможностями:
- Автоматическое определение кодировки файла
- Визуальное сопоставление колонок с полями
- Режим отладки для проверки первой строки
- Интеграция с [Scheduler](/components/scheduler) для больших файлов

### Поля товара

Настройка отображения полей на вкладке "Данные товара":
- Включение/отключение полей
- Группировка по секциям
- Изменение порядка отображения
- Создание и управление секциями

### Дополнительные поля

Создание новых полей для расширения моделей:
- msProductData — товары
- msVendor — производители
- msOrder — заказы
- msOrderAddress — адреса
- msCategory — категории

### Колонки гридов

Настройка колонок в административных таблицах:
- Видимость колонок
- Сортировка и фильтрация
- Ширина колонок
- Шаблоны отображения

### Поля модели

Управление существующими полями моделей:
- Редактирование настроек полей
- Привязка к секциям
- Настройка виджетов (xtype)
- Управление секциями
