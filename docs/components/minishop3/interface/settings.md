# Настройки магазина

Раздел настроек магазина доступен через **Extras → MiniShop3 → Настройки**.

## Вкладки настроек

Раздел содержит следующие вкладки:

| Вкладка | Описание |
|---------|----------|
| [Способы доставки](settings/deliveries) | Настройка методов доставки, стоимости, валидации |
| [Способы оплаты](settings/payments) | Настройка платёжных систем |
| [Производители](settings/vendors) | Справочник производителей товаров |
| [Связи товаров](settings/links) | Типы связей между товарами |
| [Опции](settings/options) | EAV-система дополнительных характеристик |

## Интерфейс

Все вкладки настроек построены на современном Vue 3 интерфейсе с использованием PrimeVue компонентов. Это обеспечивает:

- **Быстрый отклик** — нет перезагрузки страницы при редактировании
- **Drag & Drop сортировку** — изменение порядка элементов перетаскиванием
- **Массовые операции** — выбор нескольких записей для удаления
- **Поиск и фильтрация** — быстрый поиск по названию и описанию

## Общие возможности

### Создание записей

Нажмите кнопку **Добавить** для создания нового элемента. Откроется форма с полями, специфичными для каждого типа записи.

### Редактирование

Дважды кликните по записи в таблице или выберите её и нажмите **Редактировать**.

### Сортировка

Для изменения порядка отображения перетащите запись в нужную позицию в таблице. Новые позиции сохраняются автоматически.

### Удаление

Выберите одну или несколько записей и нажмите **Удалить**. Система запросит подтверждение перед удалением.

::: warning Важно
Удаление способа доставки или оплаты может повлиять на существующие заказы. Рекомендуется деактивировать неиспользуемые методы вместо удаления.
:::
