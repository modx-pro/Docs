# Поиск и кэш

Поиск сканирует Markdown-файлы по заголовку, пути и plain-text содержимому и
показывает результаты в выпадающем списке над документом.

Подготовленный поисковый индекс кешируется в:

```text
core/cache/mxlocdoc/
```

В ключе кэша учитывается выбранный языковой корень, поэтому у каждого языка свой индекс.

## Очистка кэша

Плагин `mxLocDocCacheClear` слушает событие MODX `OnBeforeCacheUpdate`. Когда
пользователь очищает кэш в менеджере, mxLocDoc удаляет и свой поисковый кэш.

Установите `mxlocdoc.cache_ttl` в `0`, чтобы отключить кеширование поискового индекса,
или `mxlocdoc.search_enabled` в `Нет`, чтобы выключить поиск полностью.
