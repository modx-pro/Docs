# Диагностика

## Папка документации не найдена

Проверьте `mxlocdoc.docs_path`. Предпочтительный переносимый формат:

```text
[[+corePath]]components/project-docs/
```

Итоговая папка после раскрытия пути должна существовать и быть доступной PHP для чтения.

## Картинки не отображаются

Проверьте, что путь к картинке указан относительно текущего Markdown-файла, а
расширение есть в `mxlocdoc.allowed_asset_extensions`.

## Поиск не обновляется

Очистите кэш менеджера MODX — mxLocDoc очищает свой кэш через плагин
`mxLocDocCacheClear` на событии `OnBeforeCacheUpdate`.

## Язык не появился в переключателе

Убедитесь, что код языка указан в настройке `mxlocdoc.languages`, языковая папка
(`en/`, `ru/` и т.п.) — прямой потомок `mxlocdoc.docs_path`, и внутри неё есть
Markdown-документация.
