# 2.7.1-pl

**Дата релиза:** 21 июля 2026

Патч про `task_update` в MCP.

## Исправления

- **`fields` применяется частичным патчем.** Непереданные ключи сохраняются, поэтому обновление одного поля больше не падает на уже заполненных обязательных полях типа. При одновременной смене `type` поля по-прежнему считаются полным набором для новой схемы.
- **Неизвестные ключи `fields` отклоняются** валидационной ошибкой, а не игнорируются молча с ответом «обновлено».

## Тесты

In-process API/MCP smoke получил регрессию на частичный патч `fields`, полный update с `fields`, ошибку пустого и неизвестного поля, а также на неизменность REST-контракта.
