# Словоформы

Словоформы обеспечивают склонение слов в SEO-текстах.

## Назначение

Русский язык требует склонения слов по падежам:

| Падеж | Пример |
|-------|--------|
| Именительный | телефон |
| Родительный | телефон**а** |
| Дательный | телефон**у** |
| Винительный | телефон |
| Творительный | телефон**ом** |
| Предложный | телефон**е** |

## Таблица словоформ

| Столбец | Описание |
|---------|----------|
| **Слово** | Именительный падеж |
| **Родительный** | Форма родительного падежа |
| **Дательный** | Форма дательного падежа |
| **Винительный** | Форма винительного падежа |
| **Творительный** | Форма творительного падежа |
| **Предложный** | Форма предложного падежа |
| **Множественное** | Форма множественного числа |

## Создание словоформы

### 1. Нажмите «Создать»

### 2. Заполните формы

| Поле | Пример |
|------|--------|
| **Слово** | телефон |
| **Родительный** | телефона |
| **Дательный** | телефону |
| **Винительный** | телефон |
| **Творительный** | телефоном |
| **Предложный** | телефоне |
| **Множественное** | телефоны |

### 3. Сохраните

## Использование в SEO шаблонах

### Модификаторы падежей

| Модификатор | Падеж | Пример |
|-------------|-------|--------|
| `\|nominative` | Именительный | `{$filters.vendor\|nominative}` |
| `\|genitive` | Родительный | `{$filters.vendor\|genitive}` |
| `\|dative` | Дательный | `{$filters.vendor\|dative}` |
| `\|accusative` | Винительный | `{$filters.vendor\|accusative}` |
| `\|instrumental` | Творительный | `{$filters.vendor\|instrumental}` |
| `\|prepositional` | Предложный | `{$filters.vendor\|prepositional}` |
| `\|plural` | Множественное | `{$filters.vendor\|plural}` |

### Примеры

**Шаблон title:**
```
Купить {$filters.vendor|accusative} в Москве
```

**Результат:**
- Входное значение: `Samsung`
- Словоформа: `accusative = Samsung`
- Результат: `Купить Samsung в Москве`

**Шаблон description:**
```
Большой выбор {$filters.vendor|genitive}. Доставка по России.
```

**Результат:**
- Входное значение: `телефон`
- Словоформа: `genitive = телефона`
- Результат: `Большой выбор телефона. Доставка по России.`

## Автоматическое склонение

Если словоформа не найдена в базе, система попытается:

1. Использовать слово без изменений
2. Применить базовые правила русского языка (если включено)

## Программный доступ

```php
$mfilter = $modx->services->get('mfilter');
$wordForms = $mfilter->getWordFormsManager();

// Получить форму
$genitive = $wordForms->getForm('телефон', 'genitive');
// 'телефона'

// Проверить наличие
$exists = $wordForms->hasWord('телефон');

// Добавить словоформу
$wordForms->add([
    'word' => 'ноутбук',
    'genitive' => 'ноутбука',
    'dative' => 'ноутбуку',
    'accusative' => 'ноутбук',
    'instrumental' => 'ноутбуком',
    'prepositional' => 'ноутбуке',
    'plural' => 'ноутбуки'
]);
```

## Советы

1. **Добавляйте бренды** — Samsung, Apple, Xiaomi обычно не склоняются
2. **Добавляйте категории** — телефон, планшет, ноутбук
3. **Проверяйте результат** — просмотрите сгенерированные тексты
4. **Используйте импорт** — для большого количества слов

## Частые случаи

### Несклоняемые слова

Для иностранных брендов заполните все формы одинаково:

| Поле | Значение |
|------|----------|
| Слово | Apple |
| Родительный | Apple |
| Дательный | Apple |
| ... | Apple |

### Слова с чередованием

| Дательный | дню |
| Винительный | день |
| Творительный | днём |
| Предложный | дне |
| Множественное | дни |
