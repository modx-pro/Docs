# Интерфейс админки

Административный интерфейс mFilter находится в **Установщик → mFilter**.

## Вкладки

| Вкладка | Описание |
|---------|----------|
| [Наборы фильтров](filter-sets) | Создание и настройка наборов фильтров |
| [Слаги](slugs) | SEO-алиасы для значений фильтров |
| [Паттерны URL](patterns) | Правила распознавания URL |
| [SEO шаблоны](seo-templates) | Шаблоны для генерации метаданных |
| [Словоформы](word-forms) | Склонение слов для SEO-текстов |
| [Обслуживание](maintenance) | Индекс фасетов, кэш, диагностика |
| [Scheduler-задачи](scheduler) | Список фоновых задач, интервалы, параметры |
| [Прогрев кэша](warmup) | *Legacy*: прогрев baseIds для AJAX |

## Быстрый старт

### 1. Создайте набор фильтров

1. Перейдите на вкладку **Наборы фильтров**
2. Нажмите **Создать**
3. Введите название и добавьте фильтры
4. Привяжите к категориям каталога

### 2. Настройте слаги (опционально)

1. Перейдите на вкладку **Слаги**
2. Просмотрите автоматически созданные слаги
3. Отредактируйте при необходимости

### 3. Создайте SEO шаблоны (опционально)

1. Перейдите на вкладку **SEO шаблоны**
2. Создайте шаблоны для разных комбинаций фильтров
3. Укажите условия применения

## Технологии

Интерфейс построен на:

- **Vue 3** — реактивный фреймворк
- **PrimeVue** — UI компоненты
- **VueTools** — интеграция с MODX
