# Интерфейс

## Экспорт ресурсов

![Экспорт ресурсов](https://demo.rpa-design.ru/media/impex/001.png)

Выберите родительский ресурс в дереве, конфигурацию (кнопка рядом открывает файл конфигурации для правки), тип файла и нажмите **Экспорт**. Ссылка на созданный файл появится на странице.

## Импорт ресурсов

![Импорт ресурсов](https://demo.rpa-design.ru/media/impex/002.png)

1. Выберите родительский ресурс и конфигурацию.
2. Загрузите файл или выберите его из списка (лишние файлы можно удалить).
3. Выберите режим обработки галереи miniShop3. Изображения заранее положите в директорию-источник из `$impex_config['content_row']` (тип `msgallery`).

Режимы галереи:

- **Не обрабатывать** — колонка с изображениями игнорируется.
- **Загружать только новые** — для новых товаров загружаются все файлы; для существующих старые остаются, новые из файла и источника добавляются.
- **Обновить всё** — у существующих товаров галерея очищается, затем загружаются файлы из импорта.

Операции идут через процессоры miniShop3 `gallery/upload` и `gallery/removeall`. Галерея нагружает сервер: подбирайте режим под мощность хостинга и `batch_import` в [конфигурации](configuration).

После успешной обработки галереи исходники из директории-источника удаляются.

### Дополнительные опции

| Опция | Описание |
| --- | --- |
| **Тестирование** | После импорта откроется ссылка на системный журнал с массивом данных. Новые товары, изменения от плагинов и новые категории помечены. |
| **Очищать директорию перед импортом** | Перед импортом все ресурсы в категории помечаются удалёнными (корзина MODX). Позиции из файла заливаются заново; удалённые ранее позиции из файла восстанавливаются из корзины. |
| **Обновлять поисковый индекс mSearch** | После импорта ресурс индексируется процессором mSearch, если компонент установлен. |
| **Сохранять резервную копию** | Перед импортом дамп контентных таблиц в `{assets_url}components/impex3/backups/`. Ссылка на файл выводится в интерфейсе. |

Таблицы в бэкапе (`modx_` — префикс по умолчанию):

```text
modx_site_content
modx_site_tmplvar_contentvalues
```

При установленном miniShop3 добавляются:

```text
modx_ms3_products
modx_ms3_vendors
modx_ms3_product_options
modx_ms3_product_files
modx_ms3_product_categories
```
