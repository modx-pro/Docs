# Сниппеты msRussianPost2

Дополнение предоставляет два сниппета для страницы оформления заказа: лексикон для JavaScript и подключение стилей, скрипта и конфигурации виджета.

- [msRussianPost2](msRussianPost2) — CSS, JS и объект `window.msRussianPost2Config`. Вызывать **после** `msrpLexiconScript`.
- [msrpLexiconScript](msrpLexiconScript) — выставляет `window.msrpLexicon` из лексикона MODX. Вызывать **первым**.
