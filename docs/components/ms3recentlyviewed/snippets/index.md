# Сниппеты ms3RecentlyViewed

Компонент предоставляет три сниппета для вывода блока просмотренных, похожих товаров и лексикона для фронта.

- [ms3recentlyviewed](ms3recentlyviewed) — вывод списка товаров по переданным ID (или из БД при fromDB=true)
- [ms3recentlyviewedSimilar](ms3recentlyviewedSimilar) — товары из тех же категорий, что и просмотренные («похожие на просмотренные»)
- [ms3rvLexiconScript](ms3rvLexiconScript) — лексикон и конфиг для JS (язык, maxItems); подключать **до** viewed.js
