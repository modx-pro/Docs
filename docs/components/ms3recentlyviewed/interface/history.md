# История просмотров

Вкладка **«История»** — таблица записей просмотров с фильтрами и действиями.

## Фильтры

- **По дате** — диапазон (dateFrom, dateTo)
- **По пользователю** — ID пользователя
- **По товару** — поиск по названию или ID товара

## Таблица

Колонки: `user_id`, `product_id`, `viewed_at`. Сортировка по колонкам.

## Действия

- **Удаление одной записи** — кнопка в строке
- **Массовое удаление** — чекбоксы и кнопка «Удалить выбранные»
- **Экспорт CSV** — кнопка «Экспорт»: выгрузка отфильтрованных данных (CSV, BOM UTF-8, разделитель `;`)

Для удаления и экспорта требуется право **save_log**.
