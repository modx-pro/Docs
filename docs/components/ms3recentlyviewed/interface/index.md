# Интерфейс админки

Управление и аналитика просмотров товаров в менеджере MODX.

**Вход:** **Extras → ms3RecentlyViewed**. Требуются права `view` (просмотр) и `save_log` (удаление, экспорт).

## Разделы

- [Дашборд](dashboard) — KPI-карточки и топ товаров по просмотрам
- [История просмотров](history) — таблица с фильтрами, удаление записей, экспорт CSV

Подробнее о правах: [Права доступа](../permissions).
