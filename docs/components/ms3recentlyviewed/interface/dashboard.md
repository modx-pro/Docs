# Дашборд

Вкладка **«Дашборд»** в **Extras → ms3RecentlyViewed** — сводные показатели по просмотрам товаров. Контент вкладки при необходимости прокручивается (скролл внутри области).

## KPI-карточки

- **Всего просмотров** — суммарное количество записей в истории. В подсказке: за сегодня, за неделю, за месяц (визуально различаются цветом: сегодня — основной, неделя — зелёный, месяц — серый). Отступы между блоками периодов.
- **Уникальных товаров** — число уникальных товаров, которые просматривали.
- **Уникальных пользователей** — число уникальных посетителей (user_id или session_id).
- **Среднее на пользователя** — среднее количество просмотров на одного посетителя.

Заголовки карточек выделены для лучшей читаемости.

## Топ просматриваемых товаров (Топ-10)

Таблица 10 наиболее просматриваемых товаров: порядковый номер (#), ID, название, количество просмотров. Сортировка по всем колонкам, включая #; ранг обновляется при смене порядка. Без пагинации. Кнопка **«Обновить»** перезагружает данные.

Данные берутся из таблицы `ms3recentlyviewed_items` (при включённом архивировании — из сводной таблицы `ms3recentlyviewed_monthly`).
