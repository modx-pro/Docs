# Дашборд

Вкладка **«Дашборд»** в **Extras → ms3RecentlyViewed** — сводные показатели по просмотрам товаров.

## KPI-карточки

- **Всего просмотров** — суммарное количество записей в истории
- **Пользователей** — число уникальных пользователей с записями
- **Товаров** — число уникальных товаров, которые просматривали
- **За сегодня / за неделю / за месяц** — счётчики за период (если доступны в данных)

## Топ товаров

Таблица наиболее просматриваемых товаров: название, ID, количество просмотров. Сортировка по количеству просмотров.

Данные берутся из таблицы `ms3recentlyviewed_items` (при включённом архивировании — из сводной таблицы `ms3recentlyviewed_monthly`).
