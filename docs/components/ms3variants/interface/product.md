# Вкладка вариантов в товаре

При редактировании товара MiniShop3 появляется дополнительная вкладка **Варианты**, позволяющая управлять вариантами товара.

## Общий вид

<!-- Скриншот: вкладка вариантов с таблицей -->

Вкладка содержит:
- **Статистику** — общее количество вариантов, активных вариантов и суммарные остатки
- **Таблицу вариантов** — список всех вариантов с основной информацией
- **Кнопку создания** — для добавления нового варианта

## Таблица вариантов

Таблица отображает все варианты товара:

| Колонка | Описание |
|---------|----------|
| Изображение | Миниатюра изображения варианта |
| Опции | Комбинация опций (цвет, размер и т.д.) |
| Артикул | SKU варианта |
| Цена | Текущая цена |
| Остаток | Количество на складе |
| Активность | Включен/выключен |
| Действия | Кнопки редактирования и удаления |

## Создание варианта

<!-- Скриншот: диалог создания варианта -->

1. Нажмите кнопку **Создать вариант**
2. В открывшемся диалоге заполните поля:

### Поля варианта

| Поле | Обязательное | Описание |
|------|--------------|----------|
| Опции | Да | Минимум одна опция (ключ + значение) |
| Артикул | Нет | Генерируется автоматически по шаблону |
| Цена | Да | Цена варианта |
| Старая цена | Нет | Для отображения скидки |
| Остаток | Нет | Количество на складе (по умолчанию 0) |
| Вес | Нет | Вес варианта |
| Изображение | Нет | Выбор из галереи товара |
| Активность | — | Включен по умолчанию |

### Добавление опций

1. Нажмите **Добавить опцию**
2. Выберите ключ опции из списка или введите новый
3. Введите значение опции
4. Повторите для всех нужных опций

**Стандартные ключи опций:**
- `color` — Цвет
- `size` — Размер
- `material` — Материал
- `volume` — Объём
- `weight` — Вес (как опция, не путать с полем веса)

::: tip Синхронизация с фильтрами
При сохранении варианта значения опций `color`, `size` и `tags` автоматически добавляются в поле msProductData товара. Это позволяет использовать стандартные фильтры MiniShop3 для поиска товаров по опциям вариантов.
:::

### Выбор изображения

<!-- Скриншот: выбор изображения из галереи -->

1. Нажмите на область выбора изображения
2. Откроется галерея товара
3. Выберите нужное изображение
4. Изображение будет привязано к варианту

::: warning Изображения из галереи
Для варианта можно выбрать только изображения, уже загруженные в галерею товара. Сначала добавьте все нужные изображения на вкладке **Галерея**.
:::

## Редактирование варианта

1. Нажмите на строку варианта или кнопку редактирования
2. Измените нужные поля
3. Нажмите **Сохранить**

## Удаление варианта

1. Нажмите кнопку удаления (корзина) в строке варианта
2. Подтвердите удаление

::: danger Внимание
Удаление варианта необратимо. Если вариант был в заказах, информация о нём в заказах сохранится, но связь с вариантом будет потеряна.
:::

## Массовые операции

### Быстрое изменение цены

Для быстрого изменения цены нескольких вариантов:
1. Откройте каждый вариант
2. Измените цену
3. Сохраните

### Деактивация вариантов

Если вариант временно недоступен:
1. Откройте вариант
2. Снимите галочку **Активен**
3. Сохраните

Неактивные варианты не отображаются на сайте и недоступны для добавления в корзину.
