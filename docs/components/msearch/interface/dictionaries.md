# Словари

Вкладка для управления морфологическими словарями phpMorphy.

## Назначение

Морфологические словари обеспечивают поиск по словоформам. Например, при поиске «телефоны» также будут найдены «телефон», «телефона», «телефонов» и т.д.

## Доступные словари

| Словарь | Язык |
|---------|------|
| ru | Русский |
| en | Английский |
| de | Немецкий |
| uk | Украинский |

::: info Требования
Для работы mSearch требуется **минимум один** морфологический словарь. Без словарей поиск по словоформам работать не будет.
:::

## Интерфейс

### Таблица словарей

| Колонка | Описание |
|---------|----------|
| Словарь | Название словаря |
| Язык | Язык словаря |
| Установлен | Статус установки |

### Действия

| Кнопка | Описание |
|--------|----------|
| Установить | Скачать и установить словарь |
| Удалить | Удалить установленный словарь |

## Установка словаря

1. Найдите нужный словарь в таблице
2. Нажмите **Установить**
3. Выберите зеркало для скачивания
4. Дождитесь завершения загрузки

### Источник словарей

Словари скачиваются с GitHub (репозиторий [seoservice2020/phpmorphy](https://github.com/seoservice2020/phpmorphy)).

::: tip Composer-пакет
mSearch использует библиотеку `cijic/phpmorphy` — современный форк phpMorphy, устанавливаемый через Composer.
:::

## После установки

После установки нового словаря рекомендуется:

1. Перейти на вкладку [Индексация](/components/msearch/interface/indexes)
2. Очистить существующий индекс
3. Создать индекс заново

Это обеспечит корректную обработку слов новым словарём.

## Расположение файлов

Словари хранятся в директории:
```
core/components/msearch/phpmorphy/dicts/
```

Каждый словарь представляет собой набор файлов `.mrd`, `.fsa` и `meta.json`.

## Работа без словарей

Если словари не установлены или удалены:

- Поиск продолжит работать
- Морфологический анализ будет отключён
- Поиск будет только по точным совпадениям и LIKE-запросам
- Эффективность поиска значительно снизится

## Многоязычные сайты

Для многоязычных сайтов установите все необходимые словари. mSearch автоматически определит язык слова и применит соответствующий словарь.

### Пример

На сайте контент на русском и английском:
1. Установите словарь `russian`
2. Установите словарь `english`

При поиске:
- Слово «телефоны» → обработка русским словарём
- Слово «phones» → обработка английским словарём

## Отличия от mSearch2

| Аспект | mSearch2 | mSearch |
|--------|----------|---------|
| Библиотека | Встроенный phpMorphy | Composer-пакет `cijic/phpmorphy` |
| Источник словарей | SourceForge | GitHub |
| Украинский словарь | Да | Да |
| Эстонский словарь | Да | Нет |
| Интерфейс | ExtJS | Vue 3 + PrimeVue |
| Установка | При инсталляции | Вручную через интерфейс |
