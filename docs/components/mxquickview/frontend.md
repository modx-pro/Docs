# Подключение на сайте

Для пошагового внедрения используйте [Интеграция на сайт](integration).

На этой странице краткий минимум:

1. Подключите `mxQuickView.initialize` один раз в шаблоне.
2. Добавьте триггеры с `data-mxqv-click` или `data-mxqv-mouseover`.
3. Укажите режим (`modal` или `selector`) и тип рендера (`chunk`, `snippet`, `template`).
4. Проверьте whitelist в системных настройках `mxquickview`.
5. Для MiniShop3 карточек проверьте `add-to-cart` сценарий в quick view.
