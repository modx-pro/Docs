# Сниппеты CrawlerDetect

Дополнение предоставляет два элемента для определения ботов и защиты форм.

- [isCrawler](isCrawler) — определяет, является ли текущий посетитель ботом (возвращает `"1"` или `"0"`)
- [crawlerDetectBlock](crawlerDetectBlock) — preHook для FormIt, блокирует отправку формы ботами

Оба элемента устанавливаются вместе с пакетом. Для защиты форм обязателен FormIt.
