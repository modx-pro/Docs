# TicketMeta

Мета тикета на странице: автор, даты, просмотры, голосование, избранное. С 1.4.0 работает и с обычными ресурсами MODX.

## Параметры

| Название | По умолчанию | Описание |
| --- | --- | --- |
| **&tpl** | `tpl.Tickets.meta` | Чанк блока мета |
| **&tplFile** | `tpl.Tickets.meta.file` | Чанк строки файла в списке |
| **&getUser** | `1` | Подгружать профиль автора |
| **&getFiles** | `1` | Выводить список загруженных файлов |
| **&unusedFiles** | `1` | Только файлы без ссылки в `content` |
| **&thread** | | Имя ветки комментариев для счётчиков; по умолчанию `resource-{id}` |
| **&id** | текущий ресурс | ID ресурса, если сниппет не на странице тикета |

Параметр `getSection` в окне свойств сниппета отсутствует; секция подставляется в коде при классе `Ticket`.

## Примеры

### Стандартный вызов

::: code-group

```fenom
{'!TicketMeta' | snippet}
```

```modx
[[!TicketMeta]]
```

:::

### Плейсхолдеры без чанка

::: code-group

```fenom
{'!TicketMeta' | snippet : ['tpl' => '']}
```

```modx
[[!TicketMeta? &tpl=``]]
```

:::

### Мета другого ресурса

::: code-group

```fenom
{'!TicketMeta' | snippet : ['id' => 42]}
```

```modx
[[!TicketMeta? &id=`42`]]
```

:::
