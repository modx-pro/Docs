# Экспорт данных и графиков

ms3Pulse поддерживает экспорт таблиц в CSV и графиков в PNG.

## Экспорт таблиц в CSV

Доступен на вкладке **«Топы / Флопы»**.

- **У каждой таблицы** — кнопка экспорта (CSV). Формируется файл с данными выбранной таблицы с учётом текущих фильтров (период, статусы).
- **Общий экспорт** — выгрузка всех отображаемых таблиц (в зависимости от реализации интерфейса).

Параметры вызова процессора `Dashboard/ExportCsv`:

| Параметр | Описание |
|----------|----------|
| `date_from`, `date_to` | Период. |
| `status_ids` | ID статусов через запятую (пусто = все). |
| `table_type` | `top` \| `flop` \| `orders_by_status` \| `revenue_by_day` \| `orders_by_delivery` \| `orders_by_payment`. |

Формат файла: UTF-8 с BOM для корректного открытия в Excel. Имя файла формируется автоматически (например, `ms3pulse-top-2026-02-11.csv`).

## Экспорт графика в PNG

Для каждого графика (карточка с диаграммой):

1. Откройте меню карточки (три точки).
2. Выберите **«Экспорт изображения (PNG)»**.

Изображение формируется средствами ECharts (`getDataURL`) с повышенным разрешением (`pixelRatio: 2`), фон — белый. Сохранение файла выполняется браузером.

## Экспорт данных графика в CSV

При наличии в меню карточки графика пункта **«Экспорт данных (CSV)»** — по нему выполняется выгрузка данных текущего графика в CSV.
