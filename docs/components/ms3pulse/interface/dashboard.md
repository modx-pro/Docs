# Интерфейс дашборда

Описание элементов главной страницы ms3Pulse.

## Шапка

- Название **ms3Pulse**.
- На вкладке «Топы / Флопы» — кнопки экспорта таблиц.

## Панель фильтров

По умолчанию **свёрнута**. Раскрывается по клику на заголовок **«Фильтры»**.

| Элемент | Описание |
|---------|----------|
| **Период** | Поля «От» и «До» (DatePicker, формат ДД.ММ.ГГГГ). |
| **Статусы** | Мультивыбор статусов заказов MiniShop3. Пусто = все статусы. Цвета из настроек MiniShop3. |
| **Группировка** | День / неделя / месяц — агрегация точек на графиках по времени. |
| **Пресеты** | Кнопки 7, 30, 90, 180, 365 дней — автоматическая установка периода («До» = сегодня, «От» = сегодня − N). |
| **Обновить** | Перезагрузка данных. |

## Вкладки дашборда

| Вкладка | Содержимое |
|---------|------------|
| **Обзор** | 5 карточек метрик (выручка, заказы, средний чек, маржа, товаров), сравнение с предыдущим периодом (%), мини-график выручки. |
| **Динамика** | 3 фиксированных графика (выручка, заказы, средний чек по дням) + кастомные. Кнопка «В конструктор». |
| **Товары и статусы** | Топ товаров (горизонтальные столбцы), заказы по статусам (кольцевая диаграмма) + кастомные. Кнопка «В конструктор». |
| **Топы / Флопы** | Конструктор таблиц: 6 типов данных, пагинация, экспорт CSV. |

## Пустые состояния

- Нет данных за период — сообщение с предложением изменить фильтры или период.
- Нет кастомных графиков — подсказка добавить график через конструктор.

## Интерактивность графиков

- **Tooltip** при наведении на точки/сегменты.
- **Меню карточки** (три точки): размер (100%, 75%, 50%, 25%), экспорт в PNG, экспорт данных в CSV, удаление (для кастомных).
- **Перетаскивание** — изменение порядка кастомных графиков на вкладке.
