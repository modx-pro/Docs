# Настройки в интерфейсе

Пункт меню **ms3Pulse → Настройки** открывает **Системные настройки** MODX с предустановленным фильтром по пространству имён **ms3pulse**.

Таким образом можно быстро перейти к настройкам дашборда и отчётов по расписанию без ручного поиска пространства имён.

Полное описание параметров — на странице [Системные настройки](../settings).
