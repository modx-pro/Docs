# Интерфейс ms3Pulse

Краткий обзор разделов интерфейса. Подробное описание — на соответствующих страницах.

## Точки входа

| Раздел | Доступ | Описание |
|--------|--------|----------|
| **Дашборд** | Компоненты → ms3Pulse | Главная страница: метрики, фильтры, вкладки с графиками и таблицами. |
| **Конструктор** | ms3Pulse → Конструктор | Добавление кастомных графиков на вкладки «Динамика» и «Товары». |
| **Настройки** | ms3Pulse → Настройки | Системные настройки (пространство имён ms3pulse). |

## Виды (views)

Контроллер поддерживает три вида через параметр `?view=`:

- **dashboard** (по умолчанию) — основной дашборд.
- **builder** — страница конструктора графиков.
- **topflops** — страница «Топы / Флопы» (доступна и как вкладка на дашборде).

## Документация по разделам

- [Дашборд](dashboard) — шапка, панель фильтров, вкладки, пустые состояния.
- [Конструктор графиков](builder) — форма добавления графика, превью, хранение в localStorage.
- [Топы и флопы](topflops) — типы таблиц, добавление, сортировка, пагинация, экспорт.
- [Настройки](settings) — быстрая ссылка на системные настройки ms3pulse.
