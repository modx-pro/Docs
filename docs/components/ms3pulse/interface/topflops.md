# Топы и флопы (конструктор таблиц)

Вкладка **«Топы / Флопы»** дашборда — настраиваемые таблицы с данными по заказам и товарам.

## 6 типов таблиц

| Тип | Ключ | Колонки |
|-----|------|---------|
| Топ товаров | `top` | №, Товар, Выручка, Кол-во, Маржа, % от общей выручки |
| Флоп товаров | `flop` | Те же колонки (товары с наименьшей выручкой) |
| Заказы по статусам | `orders_by_status` | Статус, Заказов, Выручка |
| Выручка по дням | `revenue_by_day` | Дата, Выручка, Заказов |
| Заказы по доставке | `orders_by_delivery` | Доставка, Заказов, Выручка |
| Заказы по оплате | `orders_by_payment` | Оплата, Заказов, Выручка |

## Форма «Добавить таблицу»

Панель по умолчанию **свёрнута**. Поля:

| Поле | Описание |
|------|----------|
| **Тип** | Один из 6 типов таблиц. |
| **Название** | Обязательно — заголовок блока таблицы. |
| **Ширина** | 50% или 100% ширины строки. |
| **Лимит строк** | 5, 10, 50, 100. |

## Возможности таблиц

- **Сортировка** — клик по заголовку колонки (по возрастанию/убыванию).
- **Пагинация** — выбор количества строк на страницу (5, 10, 50, 100) и переключатель страниц. Колонка «№» — сквозной ранг.
- **Итоговая строка** — суммы по выручке, количеству, марже где применимо.
- **Экспорт CSV** — кнопка у каждой таблицы или общая кнопка экспорта.
- **Хранение** — конфигурация таблиц в localStorage.

## Экспорт

Кнопка **CSV** у таблицы формирует файл в кодировке UTF-8 с BOM для корректного открытия в Excel. Имя файла формируется автоматически (например, `ms3pulse-top-2026-02-11.csv`).
