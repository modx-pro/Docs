# Метрики и показатели

На вкладке **«Обзор»** отображаются пять основных метрик и мини-график выручки.

## Карточки метрик

| Метрика | Расчёт | Индикатор |
|---------|--------|-----------|
| **Выручка** | Сумма стоимости заказов за период (по выбранным статусам). | % изменения к предыдущему периоду той же длины. Зелёный — рост, красный — падение. |
| **Заказов** | Количество заказов за период. | % к предыдущему периоду. |
| **Средний чек** | Выручка ÷ количество заказов. | % к предыдущему периоду. |
| **Маржа** | Сумма (цена − себестоимость) × количество по позициям заказов. Требуется поле **cost** (себестоимость) в товарах MiniShop3. | % к предыдущему периоду; также отображается доля маржи от выручки (%). |
| **Товаров** | Сумма количества позиций в заказах за период. | — |

**Предыдущий период** — отрезок той же длины, сразу перед выбранным. Пример: период 1–31 января (31 день) → предыдущий 1–31 декабря (31 день).

## Мини-график выручки

На вкладке «Обзор» под карточками выводится линейный график выручки по дням за выбранный период (с заливкой под линией). Масштаб и подписи осей подстраиваются под данные.

## Единицы и форматирование

- Валюта — символ из лексикона (по умолчанию ₽).
- Количество — «шт» (штуки).
- Числа на осях графиков форматируются с разделителем тысяч (locale ru-RU).
