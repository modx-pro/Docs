# Типы графиков

ms3Pulse использует **Apache ECharts 6**. Доступно 15 типов визуализации.

## Список типов

| Тип | Ключ | Описание |
|-----|------|----------|
| Линейный | `line` | Плавная линия по времени. |
| Ступенчатая линия | `lineStep` | Линия с дискретными шагами. |
| С областью | `area` | Линия с заливкой под кривой. |
| Область с накоплением | `stackedArea` | Площадной график с накоплением серий. |
| Столбчатый | `bar` | Вертикальные столбцы. |
| Горизонтальные столбцы | `barHorizontal` | Горизонтальные столбцы (рейтинги, топы). |
| Столбцы с накоплением | `stackedBar` | Несколько серий в столбцах с суммой. |
| Точечный | `scatter` | Точки по осям X/Y. |
| Круговая | `pie` | Круговая диаграмма (доли). |
| Роза (Nightingale) | `rose` | Круговая с секторами по площади (roseType: 'area'). |
| Кольцевая | `donut` | Круговая с вырезом в центре. |
| Воронка | `funnel` | Убывающие блоки (воронка). |
| Радар | `radar` | Радарная диаграмма. |
| Спидометр | `gauge` | Индикатор-стрелка. |
| Дерево | `treemap` | Прямоугольники по величине. |

## Палитра цветов

Используется 8 цветов по кругу: `#6CB24A`, `#3498db`, `#e74c3c`, `#f39c12`, `#9b59b6`, `#1abc9c`, `#34495e`, `#e67e22`.

## Ограничения по типам

- Для источников «по дням» (выручка, заказы, средний чек) логичны линейные, столбчатые, точечные, область.
- Для топа товаров и заказов по статусам — горизонтальные столбцы, круговая, кольцевая, воронка, дерево, радар.

Не все комбинации тип × источник дают осмысленный результат; в конструкторе можно опираться на [рекомендуемые сочетания](../interface/builder#рекомендуемые-сочетания).
