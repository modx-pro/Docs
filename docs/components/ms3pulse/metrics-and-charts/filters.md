# Фильтры и период

Панель фильтров определяет период, статусы заказов и способ группировки данных на графиках.

## Календарь (период)

- Поля **«От»** и **«До»** — выбор даты через DatePicker или ввод вручную (ДД.ММ.ГГГГ).
- В календаре доступны кнопки **«Сегодня»** и **«Очистить»**.
- Ограничения: «От» не может быть позже «До», «До» не может быть позже текущей даты.
- При наличии заказов в БД интерфейс может ограничивать доступный диапазон дат (подсказка с сервера).

## Пресеты периода

Кнопки **7**, **30**, **90**, **180**, **365** дней. При нажатии:

- **До** = сегодня;
- **От** = сегодня − N дней.

Данные перезагружаются автоматически после выбора пресета.

## Группировка

Определяет, как агрегируются точки на графиках «по дням/неделям/месяцам»:

| Значение | Описание | Рекомендуемый диапазон |
|----------|----------|-------------------------|
| По дням | Одна точка = 1 день | До ~60 дней |
| По неделям | Одна точка = 1 неделя | 1–3 месяца |
| По месяцам | Одна точка = 1 месяц | Квартал и больше |

## Статусы заказов

Мультивыбор статусов MiniShop3. Цвета берутся из настроек статусов в MiniShop3.

- **Пустой выбор** = учитываются все статусы.
- От выбора зависят метрики, графики и таблицы (выручка, заказы, топы и т.д. считаются только по выбранным статусам).

## Кнопка «Обновить»

Перезагружает данные с сервера с учётом текущих значений фильтров (период, статусы, группировка).
