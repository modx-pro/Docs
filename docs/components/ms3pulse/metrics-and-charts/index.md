# Метрики и графики

Раздел описывает показатели дашборда, фильтры периода и статусов, типы графиков и источники данных для визуализации.

- [Метрики и показатели](metrics) — карточки на вкладке «Обзор», формулы расчёта, сравнение с предыдущим периодом.
- [Фильтры и период](filters) — календарь, пресеты, группировка, выбор статусов.
- [Типы графиков](chart-types) — 15 типов (линия, столбцы, круговая, воронка, дерево и др.).
- [Источники данных](data-sources) — 5 источников для графиков (выручка по дням, заказы, топ товаров, статусы).
