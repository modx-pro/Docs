# Права доступа

Доступ к странице админки `ms3ProductSets` в менеджере MODX контролируется правом `view`.

## Права

| Право | Действие |
| --- | --- |
| `view` | Открытие страницы **Компоненты → Подборки товаров** и работа с интерфейсом |

Без права `view` страница компонента недоступна.

## Назначение

Права задаются в политиках MODX: **Управление → Права доступа**. Назначьте право `view` нужным ролям или пользователям, которые работают с подборками.
