# Интерфейс админки

Управление шаблонами подборок выполняется в менеджере MODX через страницу **Компоненты → Подборки товаров**.

Для работы интерфейса требуется установленный пакет **VueTools**.

## Разделы

- [Подборки товаров](templates) — список шаблонов, создание/редактирование, применение и отвязка от категорий

Подробнее о правах: [Права доступа](../permissions).
