# Скриншоты для «Руководство по админке» (ms3ProductSets)

Положите PNG/WebP в эту папку и подключите их в `../admin.md` вместо блоков «Место для скриншота».

| Файл (рекомендуемое имя) | Что показать |
|--------------------------|--------------|
| `admin-menu.png` | Менеджер MODX: **Компоненты → Подборки товаров** |
| `admin-page-overview.png` | Общий вид страницы: список шаблонов + панель справа (форма / дерево категорий) |
| `admin-template-list.png` | Таблица шаблонов: колонки ID, название, тип, связанные товары |
| `admin-template-form-create.png` | Форма **создания** шаблона (пустые поля или с подсказками) |
| `admin-template-form-edit.png` | Форма **редактирования** с заполненными `name`, `type`, товарами |
| `admin-product-picker.png` | Выбор товаров (пикер / поле ID), если в UI есть отдельный шаг |
| `admin-apply-category.png` | Блок **Применение к категории**: шаблон, дерево категорий, чекбокс «Заменить…», кнопка **Применить** |
| `admin-unbind.png` | **Отвязка** шаблона от категории (если вынесено отдельно в интерфейсе) |

Рекомендация: ширина **1200–1600 px**, без личных данных; при необходимости размыть домен в адресной строке браузера менеджера.

После добавления файла в `admin.md` замените блок цитаты на стандартную разметку:

```markdown
![Краткая подпись](images/admin-menu.png)
```
