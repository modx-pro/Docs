# Руководство по админке

## Раздел в manager

- Меню: **Компоненты -> Подборки товаров**
- Controller: `namespace=ms3productsets`, `action=index`
- Требуется установленный `VueTools`.

## Что можно сделать на странице

1. Создать шаблон подборки:
   - имя;
   - тип (`buy_together`, `similar`, `popcorn`, `cart_suggestion`, `vip`);
   - список товаров (через пикер или ID строкой);
   - описание (`description`, для справки в админке);
   - порядок (`sortorder`).
2. Редактировать/удалять шаблоны.
3. Применять шаблон к категории (включая вложенные подкатегории).
4. Отвязывать шаблон от категории.

## Применение шаблона к категории

1. Выберите шаблон.
2. Выберите категорию в дереве.
3. Опционально включите `Заменить существующие подборки этого типа` (`replace=true`).
4. Нажмите `Применить`.

Результат: создаются записи в `ms3_product_sets` с заполненным `template_name`.

## Отвязка шаблона

- Удаляются только связи, созданные этим шаблоном (`type + template_name`).
- Ручные TV-связи и другие шаблоны остаются.

## TV-режим на карточке товара

Поддерживаемые TV:

- `ms3productsets_buy_together`
- `ms3productsets_similar`
- `ms3productsets_popcorn`
- `ms3productsets_cart_suggestion`
- `ms3productsets_vip`

При сохранении товара TV автоматически синхронизируются в `ms3_product_sets`.

В интерфейсе админки доступны только типы для шаблонов: `buy_together`, `similar`, `popcorn`, `cart_suggestion`, `vip`. Типы `auto`, `auto_sales`, `also-bought`, `cross-sell`, `custom` используются только в вызове сниппета (авто-рекомендации без шаблона).

## Типовой рабочий процесс

1. Для массовой логики используйте шаблоны + применение к категориям.
2. Для точечных исключений на конкретном товаре редактируйте TV поля.
3. Проверяйте фронтовый вывод соответствующим вызовом `ms3ProductSets`.

## Полная пошаговая инструкция

Детальная инструкция по управлению подборками, чек-лист проверки на стенде и ответы на частые вопросы — в файле **ADMIN-SETS.md** в установленном пакете: `core/components/ms3productsets/docs/ADMIN-SETS.md` (разделы 11–12: чек-лист проверки, частые вопросы).
