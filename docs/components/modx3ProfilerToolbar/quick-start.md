# Быстрый старт

За 3 шага: включить Dev Toolbar и увидеть метрики запроса.

## Требования

| Требование | Версия |
|------------|--------|
| MODX Revolution | 3.x |
| PHP | 8.1+ |

## Шаг 1: Установка

1. Перейдите в **Пакеты → Установщик**.
2. Найдите **Modx3ProfilerToolbar** в списке.
3. Нажмите **Установить**.

## Шаг 2: Настройка

1. Откройте **Управление → Системные настройки**.
2. Фильтр по пространству имён **modx3profilertoolbar**.
3. Включите:
   - **modx3profilertoolbar.enabled** — Да
   - **modx3profilertoolbar.frontend_toolbar** — Да

## Шаг 3: Проверка

1. Войдите в Manager (админку).
2. Откройте любую страницу сайта в контексте **web**.
3. В правом нижнем углу появится панель Dev Toolbar с метриками запроса.

Тулбар показывается только пользователям, залогиненным в Manager. Трейсы не сохраняются — данные показываются только для текущего запроса.
