# Справочник атрибутов разметки

Все атрибуты разметки `data-mpc-*` одним списком. Подробное описание с примерами «вход → выход» — по ссылкам в разделе [Разметка вёрстки](markup).

| Атрибут              | Описание                                                                                                                    | Где в разметке                                                                                                                                                                                                                 |
|----------------------|-----------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `data-mpc-section`   | Помечает смысловой блок страницы — секцию (машинное имя в значении).                                                        | [Секция](markup#секция)                                                                                                                                                                                                        |
| `data-mpc-name`      | Человекочитаемое имя секции для админки.                                                                                    | [Секция](markup#секция)                                                                                                                                                                                                        |
| `data-mpc-static`    | Секция статическая — одинаковая на всех страницах.                                                                          | [Секция](markup#секция), [Статические секции](static-sections)                                                                                                                                                                 |
| `data-mpc-copy`      | Секция — копия другой (свой контент при общей вёрстке). На чанке — не нарезать файл, только подключить (переиспользование). | [Секция](markup#секция), [Повторное использование чанка](markup#повторное-использование-чанка), [Копирование секций](copy-sections)                                                                                            |
| `data-mpc-lexicon`   | Префикс лексикона секции (группировка переводов; развести оригинал и копию).                                                | [Дополнительные атрибуты секции](markup#дополнительные-атрибуты-секции)                                                                                                                                                        |
| `data-mpc-field`     | Редактируемое поле секции (текст, ссылка, медиа, список). Имя `bg_img` — фоновое изображение.                               | [Простые поля](markup#простые-поля), [Медиа-поля](markup#медиа-поля), [Списки](markup#списки)                                                                                                                                  |
| `data-mpc-ftype`     | Тип поля/TV (text, richtext, number, date, listbox…).                                                                       | [Тип поля](markup#тип-поля), [Комбинации с TV](markup#комбинации-с-tv)                                                                                                                                                         |
| `data-mpc-values`    | Варианты для полей с выбором (listbox/option/checkbox); формат `Подпись==ключ`, список или `@SELECT`.                       | [Поля с выбором](markup#поля-с-выбором-data-mpc-values), [Комбинации с TV](markup#комбинации-с-tv)                                                                                                                             |
| `data-mpc-fcap`      | Подпись поля/TV в гриде админки.                                                                                            | [Тип поля](markup#тип-поля), [Комбинации с TV](markup#комбинации-с-tv)                                                                                                                                                         |
| `data-mpc-fdesc`     | Описание поля/TV в гриде админки.                                                                                           | [Тип поля](markup#тип-поля), [Комбинации с TV](markup#комбинации-с-tv)                                                                                                                                                         |
| `data-mpc-item`      | Повтор в списке (элемент MIGX-грида). Индекс уровня — `data-mpc-item-1` и т.д.                                              | [Списки](markup#списки), [Список в списке](markup#список-в-списке-двухуровневый)                                                                                                                                               |
| `data-mpc-max`       | Лимит числа строк списка в админке (MIGX `maxRecords`).                                                                     | [Типы и атрибуты внутри списка](markup#типы-и-атрибуты-внутри-списка)                                                                                                                                                          |
| `data-mpc-lim`       | Лимит элементов списка при выводе на фронте (`{foreach}` limit).                                                            | [Типы и атрибуты внутри списка](markup#типы-и-атрибуты-внутри-списка)                                                                                                                                                          |
| `data-mpc-off`       | Смещение (offset) при выводе списка на фронте.                                                                              | [Типы и атрибуты внутри списка](markup#типы-и-атрибуты-внутри-списка)                                                                                                                                                          |
| `data-mpc-cond`      | Условие вывода элемента списка по номеру итерации (`$i`/`$l`).                                                              | [Типы и атрибуты внутри списка](markup#типы-и-атрибуты-внутри-списка)                                                                                                                                                          |
| `data-mpc-if`        | Условный вывод элемента (поле/секция/сниппет): оборачивает в `{if …}`.                                                      | [Другие атрибуты поля](markup#другие-атрибуты-поля), [Дополнительные атрибуты секции](markup#дополнительные-атрибуты-секции), [Сниппеты и чанки](markup#сниппеты-и-чанки), [Условный вывод (настройки)](markup#условный-вывод) |
| `data-mpc-remove`    | Собрать значение в поле, но сам элемент убрать из вёрстки (служебные данные, inline-стили).                                 | [Другие атрибуты поля](markup#другие-атрибуты-поля)                                                                                                                                                                            |
| `data-mpc-unwrap`    | Вывести только содержимое элемента, без тега-обёртки (несколько тегов в одно поле; текст ссылки; результат сниппета).       | [Базовая обёртка](markup#базовая-секция-обёртка-wrapper), [Редактируемый текст ссылки](markup#редактируемый-текст-ссылки), [Другие атрибуты поля](markup#другие-атрибуты-поля), [Сниппеты и чанки](markup#сниппеты-и-чанки)    |
| `data-mpc-snippet`   | Вызов сниппета на месте элемента для передачи имени сниппета и пресета.                                                     | [Сниппеты и чанки](markup#сниппеты-и-чанки)                                                                                                                                                                                    |
| `data-mpc-chunk`     | Вырезать содержимое в отдельный чанк (путь к файлу).                                                                        | [Сниппеты и чанки](markup#сниппеты-и-чанки), [Повторное использование чанка](markup#повторное-использование-чанка)                                                                                                             |
| `data-mpc-include`   | Подключить вырезанный чанк через `{include 'file:…'}`.                                                                      | [Сниппеты и чанки](markup#сниппеты-и-чанки)                                                                                                                                                                                    |
| `data-mpc-parse`     | Подключить чанк через `parseChunk(…)` с параметрами.                                                                        | [Сниппеты и чанки](markup#сниппеты-и-чанки)                                                                                                                                                                                    |
| `data-mpc-symbol`    | Символ парсинга (`{` — при нарезке/сохранении, `##` — отложить на фронт); для сниппетов/parse и условий по переводу.        | [Сниппеты и чанки](markup#сниппеты-и-чанки), [Другие атрибуты поля](markup#другие-атрибуты-поля)                                                                                                                               |
| `data-mpc-tv`        | Записать значение в TV ресурса (создаёт/синхронизирует TV).                                                                 | [TV-поля](markup#tv-поля), [Комбинации с TV](markup#комбинации-с-tv)                                                                                                                                                           |
| `data-mpc-rfield`    | Записать значение в нативное поле ресурса (`content`, `longtitle`…).                                                        | [Поля ресурса](markup#поля-ресурса)                                                                                                                                                                                            |
| `data-mpc-res`       | Обёртка: поля внутри относятся к другому ресурсу (id в значении) — редактирование cross-resource.                           | [Редактирование полей другого ресурса](markup#редактирование-полей-другого-ресурса-cross-resource)                                                                                                                             |
| `data-mpc-rid`       | id ресурса для вывода поля чужого ресурса (модификатор `resource:`).                                                        | [Редактирование полей другого ресурса](markup#редактирование-полей-другого-ресурса-cross-resource)                                                                                                                             |
| `data-mpc-contact`   | Блок контакта для передачи типа и расположения                                                                              | [Контакты](markup#контакты)                                                                                                                                                                                                    |
| `data-mpc-cfield`    | Поле контакта (роль: `value`/`caption`/`attributes`/`fvalue`).                                                              | [Контакты](markup#контакты)                                                                                                                                                                                                    |
| `data-mpc-key`       | Устойчивый ключ контакта (идентификатор).                                                                                   | [Контакты](markup#контакты)                                                                                                                                                                                                    |
| `data-mpc-translate` | Какие под-поля контакта переводятся (переопределяет настройку).                                                             | [Перевод контактов](markup#перевод-контактов)                                                                                                                                                                                  |
| `data-mpc-info`      | Записать значение в системную/контекстную настройку MODX (или ClientConfig).                                                | [Системные настройки](markup#системные-настройки)                                                                                                                                                                              |
| `data-mpc-ctx`       | Контекст для `data-mpc-info`/`data-mpc-tv` (системная vs контекстная настройка).                                            | [Системные настройки](markup#системные-настройки)                                                                                                                                                                              |
| `data-mpc-nothumb`   | Отключить обрезку у картинки.                                                                                               | [Работа с медиа](media)                                                                                                                                                                                                        |
| `data-mpc-nolazy`    | Отключить ленивую загрузку у картинки.                                                                                      | [Работа с медиа](media)                                                                                                                                                                                                        |
| `data-mpc-thumb`     | Свои параметры обрезки у картинки.                                                                                          | [Работа с медиа](media)                                                                                                                                                                                                        |
