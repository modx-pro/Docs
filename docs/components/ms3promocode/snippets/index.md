# Сниппеты

ms3PromoCode предоставляет один сниппет — форму ввода промо-кода для покупателя.

## Обзор

| Сниппет                                  | Назначение                                    |
|------------------------------------------|-----------------------------------------------|
| [ms3PromoCodeForm](ms3promocodeform)     | Форма ввода/применения/снятия промо-кода      |

## Применение

В шаблоне страницы корзины:

```fenom
{'!ms3PromoCodeForm' | snippet}
```

Сниппет рендерит готовую форму с двумя состояниями («поле ввода» и «код применён»). При наличии применённого к корзине кода SSR сразу отдаёт состояние «применён» — без мигания формы при загрузке страницы.

UI-логика (применить, снять, обработка ошибок) и стили подключаются автоматически плагином через системную настройку `ms3promocode.frontend_assets_preset`.

Подробнее: [ms3PromoCodeForm →](ms3promocodeform)
