## ms3PromoCode 1.0.0-beta1

Первый публичный релиз компонента. Полнофункциональная система промо-кодов для MiniShop3:

- Скидки в процентах и фиксированных суммах, на всю корзину или на подпадающие позиции.
- Учёт скидки в позициях заказа — корректные суммы в чеках 54-ФЗ, email и личном кабинете без правок шаблонов.
- Богатые правила: товары, категории, бренды, универсальный фильтр по полям, фильтр по опциям.
- Современная Vue 3 + PrimeVue 4 админка: список с фильтрами, редактор кода с тремя вкладками, аналитика.
- Вкладка «Промо-код» в карточке заказа с автопересчётом при изменении состава.
- Готовая SSR-форма для покупателя + headless JS API.
- Автоматический откат применения при отмене заказа.
