# Интерфейс админки

Админка ms3PromoCode построена на Vue 3 + PrimeVue 4 и состоит из трёх разделов:

| Раздел                                  | Описание                                                                |
|-----------------------------------------|-------------------------------------------------------------------------|
| [Управление кодами](codes)              | Список кодов, создание, редактирование, генерация по маске, фильтры     |
| [Аналитика](analytics)                  | KPI и журнал применений с фильтрами                                     |
| [Вкладка в заказе](order-tab)           | Применить / снять код у конкретного заказа из карточки заказа           |

## Где находится

В дереве админ-меню MODX появляется раздел **Промо-коды** с двумя пунктами:

- **Управление кодами** — список кодов и редактор.
- **Аналитика** — KPI и журнал.

Третий интерфейс — **вкладка «Промо-код»** — встраивается прямо в окно редактирования заказа MS3 (раздел **Магазин → Заказы → конкретный заказ**).

## Особенности

- **Современный UI** — PrimeVue 4 компоненты, поддержка тёмной темы, отзывчивый дизайн.
- **Lazy-загрузка** — DataTable работает с серверной пагинацией и сортировкой, большие списки не тормозят.
- **Inline-действия** — кнопки «Включить/выключить», «Редактировать», «Удалить» прямо в строке таблицы.
- **Групповые действия** — выберите несколько кодов чекбоксами и примените действие (включить, выключить, удалить).
- **Подтверждения** — критичные действия (удаление, снятие кода) подтверждаются через нативный PrimeVue ConfirmDialog.

## Требования

Vue-админка работает только при установленном [VueTools](/components/vuetools/). Без него вы увидите сообщение об отсутствующей зависимости.
