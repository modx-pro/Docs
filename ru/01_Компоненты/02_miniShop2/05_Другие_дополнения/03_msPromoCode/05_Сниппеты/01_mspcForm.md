# mspcForm

Основной сниппет компонента, который реализует вывод формы для применения промо-кода на фронте.

## Параметры

| Название | По умолчанию   | Описание           |
| -------- | -------------- | ------------------ |
| **tpl**  | `tpl.mspcForm` | Чанк вывода формы. |

## Пример

```php
{'!mspcForm' | snippet}
```
